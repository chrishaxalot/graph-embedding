edges_to_nodes(graph: Graph) → Graph[source]

Generates a new graph where each edge (u, v) is replaced with two edges (u, uv), (uv, v) and a new node uv. The attributes of the edge are copied to the new edge. This method may help if a graph embedding does not not support edge attributes.
